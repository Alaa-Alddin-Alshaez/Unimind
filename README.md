# UniMind

UniMind is an AI-powered personalized academic workspace and student support system designed to help students organize learning activities, access academic support, and receive intelligent assistance through an integrated AI agent.

## Academic Scope

This project supports three university subjects:

1. System Analysis and Design
2. Human-Computer Interaction
3. Artificial Intelligence

## Planned Technology Stack

- Frontend mobile application: Flutter
- Backend REST API: Python FastAPI
- Database: PostgreSQL
- ORM: SQLAlchemy
- Authentication: JWT
- AI support: LLM-based AI Agent with tool calling

## High-Level Architecture

UniMind will follow a client-server architecture. The Flutter mobile application will communicate with a FastAPI backend through REST APIs. The backend will manage authentication, academic workspace features, relational data through PostgreSQL and SQLAlchemy, and AI-assisted workflows through an LLM-based agent layer.
